# -flask-
基于flask   jinjia2开发的前后端不分离个人项目


# 第一个项目

## 功能实现：

**1.用户注册  可以通过发送验证码来注册用户**

**2.用户登录 和登陆限制  未登录用户不得写个人博客**

**3.主页面和博客详情页面  发布博客包括作者和时间**

**4.评论功能 已注册用户可以在博客下面评论**

**5.搜索功能  可以通过搜索框搜索自己写过的博客 且按时间从大到小排序**
